# Oscilloscope Data Analyzer

一个基于 Vite + React + TypeScript 的轻量级网页应用，用于加载 **Yokogawa 示波器（Xviewer）** 导出的 CSV 波形文件，并在浏览器中还原出示波器风格的波形显示、缩放平移与光标测量。

在线预览：[https://kyriesis.github.io/Oscilloscope-Data-Analyzer/](https://kyriesis.github.io/Oscilloscope-Data-Analyzer/)

## 功能特性

- 上传或拖拽 CSV 波形文件，自动识别并解析 Yokogawa Xviewer、Rigol 等品牌示波器导出的格式。
- 根据 `HResolution` 与 `HOffset` 还原真实时间轴。
- 深色示波器界面：黑色画布、网格、彩色波形、左侧 Y 轴标识。
- 右侧通道选择面板：
  - 勾选显示/隐藏通道。
  - 拖拽通道标签调整排列顺序。
  - 双击自定义名称区域添加/修改别名（如 `Pawl SW`），固定通道名 `CH1`/`CH2` 等不可修改。
  - 点击标签右侧方形 **R** 按钮切换通道反相，波形与测量值同步取反。
- **序列锁定**：开启后记忆通道排序、Y 轴偏移、Y 轴缩放、反相状态及自定义名称到 `localStorage`；加载相同通道组的新文件时自动沿用；新文件通道不匹配时自动关闭锁定，防止旧记忆被覆盖。
- 画布左上角显示温度/电压信息（测试工况输入为空时自动隐藏）；单通道模式下显示对应的 `[V/div]`，计算已乘以通道总数并与左侧刻度保持一致。
- 画布右上角显示当前加载的 CSV 文件名。
- **Zoom X**：激活后滚轮以鼠标位置为中心缩放时间轴。
- **Zoom Y**：激活后首次点击通道仅选中，再次点击放大 Y（最大 32×）；右键单击缩小（最小 1/64）；按住左键上下拖动可平移该通道。重排通道标签后统一重置 `yOffset = 0`，避免错位。
- 拖动画布平移时间轴时鼠标变为 `grabbing` 抓取手。
- 点击通道曲线可在左侧 Y 轴显示该通道的固定名称、自定义名和 0 位标识；在横向/纵横测量或 Zoom Y 模式下点击曲线可激活目标通道。
- 光标测量：
  - **纵向光标 A/B**：`Ctrl+左键` 设置 A，`Ctrl+右键` 设置 B，可拖动虚线或标注。
  - **横向光标 C/D**：先激活目标通道，`Ctrl+左键` 设置 C，`Ctrl+右键` 设置 D。
  - **纵横光标 EF/GH**：先激活目标通道，`Ctrl+左键` 设置十字线 EF，`Ctrl+右键` 设置 GH；可拖动单线或交点移动十字线。
- 测量标注智能避让：A/B 间距不足时自动移到右侧，带半透明黑色背景框，标注下方延伸虚线。
- 测量线统一为 **0.8** 线宽，横向/纵向与纵横模式一致；所有测量光标及标签统一为醒目的霓虹粉色 `#ff2a6d`。
- X 轴时间单位自动切换：当时基 ≤ 100 ms/div 时显示为 `ms`，时基 < 1 ms 时自动切换为 `μs`，减少 scientific notation 阅读负担。
- Y 轴 `[V/div]` 值圆整到整数，不再显示小数位。
- 测量结果面板优化：
  - 横向/纵横模式下通道名与测量值分两行显示。
  - 纵向模式下 ΔT 后显示 `1/ΔT = xxx.xxx Hz`，频率单位随 ΔT 大小自动切换（如 kHz/MHz）。
  - 纵横模式下按四行显示：`E F`、`G H`、`ΔX ΔY`、`1/ΔX`。

## 本地运行

1. 安装依赖（需先安装 [Node.js](https://nodejs.org/)）：

```bash
npm install
```

2. 启动开发服务器：

```bash
npm run dev
```

3. 在浏览器中打开终端显示的地址（通常是 `http://localhost:5173/`）。

## 构建生产版本

```bash
npm run build
```

构建产物位于 `dist/` 文件夹，可部署到任意静态服务器或 GitHub Pages。

## 使用说明

1. 点击“上传或拖拽 CSV”按钮选择文件，或将 CSV 文件拖入波形显示区。
2. 在右侧“通道”面板中：
   - 点击复选框显示/隐藏通道。
   - 拖动通道标签调整排列顺序。
   - 双击通道名区域自定义名称（`CH1`/`CH2` 等固定名不可改）。
   - 点击标签右侧的 **R** 按钮切换通道反相。
3. 点击 **🔒 序列锁定** 可记忆当前通道排序、Y 偏移、Y 缩放、反相与自定义名；新文件通道不匹配时会自动关闭锁定。
4. 点击顶部 **Zoom X** 开关，激活后用鼠标滚轮缩放时间轴。
5. 点击顶部 **Zoom Y** 开关，激活后：
   - 点击某条通道首次仅选中，再次点击放大一倍。
   - 右键单击缩小一倍。
   - 按住左键上下拖动可平移该通道 Y 位置。
6. 在任意模式下按住鼠标左键拖动画布可平移时间轴。
7. 点击通道曲线可在左侧 Y 轴显示该通道标识；在测量/Zoom Y 模式下点击可激活通道。
8. 使用测量功能：
   - **纵向光标**：`Ctrl+左键` 设置 A，`Ctrl+右键` 设置 B。
   - **横向光标**：先激活通道，`Ctrl+左键` 设置 C，`Ctrl+右键` 设置 D。
   - **纵横光标**：先激活通道，`Ctrl+左键` 设置 EF，`Ctrl+右键` 设置 GH。
9. 点击 **清除光标** 可清除当前测量模式的光标与标注；点击 **重置视图** 恢复缩放、平移和测量状态（不影响序列锁定记忆）。

## 支持的数据格式

当前支持以下 CSV 格式：

### Yokogawa Xviewer

典型表头包含：

```csv
"Model","Xviewer"
"BlockNumber","1"
"TraceName","CH1","CH2","CH4","CH6",
BlockSize,10010,10010,10010,10010,
VUnit,V,A,V,V,
HResolution,1.000000e-03,...
HOffset,-1.000900e+01,...
HUnit,s,s,s,s,
```

### Rigol

典型表头：

```csv
Time(s),CH1V,CH2V,CH3V,CH4V
-1.000000e+01,-9.380000e-03,-3.666667e-03,-2.666667e-02,6.186667e-02
```

第一列为时间（秒），后续每列对应一个通道的电压值。

后续将逐步扩展 Siglent、Keysight、Tektronix 等品牌格式。
