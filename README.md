# Oscilloscope Data Analyzer

一个基于 Vite + React + TypeScript 的轻量级网页应用，用于加载 **Yokogawa 示波器（Xviewer）** 导出的 CSV 波形文件，并在浏览器中还原出示波器风格的波形显示、缩放平移与光标测量。

## 功能特性

- 上传或拖拽 Yokogawa CSV 波形文件
- 自动解析多通道数据（CH1 / CH2 / CH4 / CH6 等）
- 根据 `HResolution` 与 `HOffset` 还原真实时间轴
- 深色示波器界面：黑色画布、网格、彩色波形
- 右侧通道选择面板，独立控制每个通道的显示/隐藏，支持拖拽排序
- 顶部 Zoom X 开关：激活后滚轮缩放时间轴
- 顶部 Zoom Y 开关：激活后单通道 Y 方向操作（拖动平移、左键放大、右键缩小）
- 拖动画布平移时间轴
- 点击通道曲线显示该通道的左侧标识（通道名 + 0 位），避免多通道标识重叠
- 窗口大小变化时自动重绘

## 本地运行

1. 安装依赖（需先安装 [Node.js](https://nodejs.org/)）：

```bash
npm install
```

2. 启动开发服务器：

```bash
npm run dev
```

3. 在浏览器中打开终端显示的地址（通常是 `http://localhost:5173/`）。

## 构建生产版本

```bash
npm run build
```

构建产物位于 `dist/` 文件夹，可部署到任意静态服务器或 GitHub Pages。

## 使用说明

1. 点击“上传或拖拽 CSV”按钮选择文件，或将 CSV 文件拖入波形显示区。
2. 在右侧“通道”面板中点击复选框显示/隐藏通道；拖动通道标签可调整排列顺序；双击自定义名称区域可添加/修改自定义名称（如 `Pawl SW`），固定通道名 `CH1`/`CH2` 等不可修改。
3. 点击顶部 **Zoom X** 开关，激活后用鼠标滚轮缩放时间轴。
4. 点击顶部 **Zoom Y** 开关，激活后鼠标悬停在曲线上会显示 `ns-resize` 光标：按住左键上下拖动可移动该通道曲线，左键单击选中该通道并放大一倍，右键单击缩小一倍。
5. 在任意模式下，点击某条通道曲线即可在左侧 Y 轴显示该通道的固定名称、自定义名称和 0 位标识；拖动该通道时标识会跟随曲线移动。
6. 按住鼠标左键拖动画布平移时间轴。
7. （光标测量模块后续重新开发，当前版本暂不启用。）

## 支持的数据格式

当前优先支持 **Yokogawa Xviewer** 导出的 CSV，典型表头包含：

```csv
"Model","Xviewer"
"BlockNumber","1"
"TraceName","CH1","CH2","CH4","CH6",
BlockSize,10010,10010,10010,10010,
VUnit,V,A,V,V,
HResolution,1.000000e-03,...
HOffset,-1.000900e+01,...
HUnit,s,s,s,s,
```

后续可扩展支持更通用的两列 CSV 格式。
